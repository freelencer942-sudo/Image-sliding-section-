# Image-sliding-section-
A simple, responsive image gallery that scrolls automatically. This project uses HTML and CSS to create a continuous loop of nature images.
# How it's Organized
index.html: Sets up the structure of the slider and holds the images.
css.css: Handles the layout, colors, and the sliding movement.
Resource/: A folder containing the nature photos (forests, mountains, and waterfalls).
# Key Features
Automatic Scroll: The images move from right to left on their own using CSS animations.
Infinite Loop: The images are doubled in the code so the scroll never seems to end.
Mobile Friendly: The layout automatically adjusts its size for phones and tablets.
No Coding Needed: Works immediately in any web browser without extra plugins.
# How to Use
Download the project files.
Keep the Resource folder and css.css in the same place as the index.html file.
Open index.html to see your sliding gallery in action!
